一、项目介绍
---
基于Python实现的小红书话题情感分析系统，该系统对小红书平台的现有话题进行爬虫，对评论内容进行情感分析，主要功能如下。


### 管理员：
- 基本操作：登录、查看系统内部数据（进行情感占比分析以及数据概览）
- 爬虫模块：手动获取某一条话题下方的评论内容，并存入数据库中
- 评论分析模块：获取评论标题、词云图、以及情感分析柱状图和情绪占比图
